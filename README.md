# Simply Awesome Game Engine

## Dependencies

- glfw
- Vulkan
- C99 and C++03 compatible compiler
- meson (Build system)

## Build

First, recursively copy the repository.

Than you should just launch a simple python script, and everything should be compiled.

```sh
python compile.py
```
